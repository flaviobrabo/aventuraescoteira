# 🏕️ Aventura Escoteira

Bem-vindo ao repositório do site Aventura Escoteira! Este é um site com CMS integrado que funciona como WordPress.

## 🚀 Como usar

### Editar conteúdo (Como WordPress)
1. Acesse: https://flaviobrabo.github.io/aventuraescoteira/admin/
2. Faça login com sua conta GitHub
3. Clique em "Blog Posts"
4. Crie, edite ou delete posts
5. As mudanças são salvas automaticamente no repositório

### Acessar o site
https://flaviobrabo.github.io/aventuraescoteira/

## 📝 Estrutura do Projeto

- `index.html` - Página inicial
- `style.css` - Estilos visuais
- `script.js` - Funcionalidades
- `posts.json` - Armazena os posts
- `admin/` - Painel de administração

## 🔧 Configurações

Todos os arquivos estão no repositório GitHub. Não precisa de servidor especial!
